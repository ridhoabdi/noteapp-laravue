# Note App Laravue
NoteApp Laravue adalah aplikasi berbasis web yang menggabungkan Laravel dan Vue untuk menyediakan fitur pencatatan.

## Tech Stack
- <b>Backend :</b> Laravel
- <b>Frontend :</b> Vue.js
- <b>Database :</b> MySQL
- <b>Pengujian API :</b> Postman
  
## Fitur Aplikasi
- Membuat catatan baru
- Mengedit catatan
- Menghapus catatan
- Mencetak catatan dalam format PDF
